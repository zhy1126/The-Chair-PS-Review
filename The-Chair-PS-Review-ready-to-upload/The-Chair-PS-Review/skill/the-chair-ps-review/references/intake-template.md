## Intake rules

- Require the draft, application category, essay type, university, programme, official programme information, exact prompt, and word limit for a complete programme-specific diagnosis.
- Treat CV/background, intended direction, current word count, draft stage, and review focus as recommended context.
- If an official link is supplied and web access is available, verify current programme claims against the official source. Do not infer current modules or requirements from memory.
- If required fields are missing, list them and ask the user to complete this template. If the user wants immediate feedback, perform only a provisional text-only precheck and label programme match and task compliance `not assessable` where appropriate.
